### What we currently understand

- Per LOA, the project's source code will live in GitHub or GitLab, and Electric Citizen provides access to IIE on request.
- Default if not addressed: the repository lives in Electric Citizen's GitHub organization with IIE granted access.

### Questions for you

**K1. Does IIE have a GitHub or GitLab organization you'd like the project repository to live in, or should we host it in Electric Citizen's GitHub organization with IIE granted access?** Either path works — repositories can be transferred between organizations later if needs change (e.g., moving to a new vendor, IIE setting up its own org mid-project, or end-of-contract handover).

- **Recommendation:** If IIE does not have an existing GitHub/GitLab organization, we recommend starting on our account (it can be transferred at any time should you set one up). GitHub and similar services charge a small amount "per seat," so hosting your own repository would be an additional expense.
